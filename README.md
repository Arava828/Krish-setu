# 🌾 Krishi Setu - Smart Platform for Farmers and Buyers

A full-stack agricultural marketplace connecting farmers directly with buyers across India. Built as part of Utimiz internship project.

## 🚀 Tech Stack
- Frontend: React.js + Tailwind CSS + Vite
- Backend: Node.js + Express.js
- Database: MongoDB Atlas
- Authentication: JWT

## ✅ Phase 1 Features
- Farmer and Buyer registration and login
- Crop listing management for farmers
- Browse and search crops for buyers
- Farmer dashboard with KPIs and stats
- Buyer dashboard with order tracking
- Weather API integration
- AI price recommendation UI

## 🛠️ How to Run Locally

### 1. Clone the repo
git clone https://github.com/Arava828/Krish-setu.git

### 2. Run Backend
cd backend
npm install
npm run dev

### 3. Run Frontend
cd frontend
npm install
npm run dev

### 4. Open browser
http://localhost:5173
